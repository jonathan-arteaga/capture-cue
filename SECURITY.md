# Security Policy

CaptureCue is early-stage software. Please report security concerns privately
through GitHub Security Advisories for this repository.

## Scope

CaptureCue requests macOS permissions for screen recording, accessibility,
microphone access, and camera access. Issues related to those permissions, local
recording data, project files, or export handling are in scope.

## Supported Versions

Only the current `main` branch is supported while the app is in early
development.
